default fallback
================

    >>> import re2
    >>> re2.set_fallback_notification(re2.FALLBACK_QUIETLY)

This pattern builds with re2

    >>> pattern = re2.compile(r"foo")
    >>> type(pattern)
    <class 're2.Pattern'>

This pattern builds with default fallback module (re)

    >>> fallback_pattern = re2.compile(r"foo(?!bar)")
    >>> type(fallback_pattern)
    <class 're2.FallbackPattern'>
    >>> type(fallback_pattern._pattern)
    <class 're.Pattern'>

regex fallback
==============

    >>> import regex
    >>> re2.set_fallback_module(regex)

This pattern builds with re2

    >>> pattern = re2.compile(r"foo")
    >>> type(pattern)
    <class 're2.Pattern'>

This pattern builds with the desired fallback module (regex)

    >>> fallback_pattern = re2.compile(r"foo(?!bar)")
    >>> type(fallback_pattern)
    <class 're2.FallbackPattern'>
    >>> type(fallback_pattern._pattern)
    <class '_regex.Pattern'>

    >>> import re
    >>> re2.set_fallback_module(re)
    >>> re2.set_fallback_notification(re2.FALLBACK_QUIETLY)
